ZEBRA TECHNOLOGIES set_config — RFD40 / RFD90 IoT Connector API Reference

set_config

Command: set_config

Description

Command issued to set the complete device configuration such as current region details, ethernet & wifi configuration, end point configuration, event and alert configuration

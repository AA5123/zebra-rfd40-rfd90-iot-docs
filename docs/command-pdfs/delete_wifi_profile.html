ZEBRA TECHNOLOGIES delete_wifi_profile — RFD40 / RFD90 IoT Connector API Reference

delete_wifi_profile

Command: delete_wifi_profile

Description

Command issued to delete saved wifi profile

Request

Request Fields

FieldTypeRequiredDescription
commandstringYesSpecifies the operation being performed, in this case, deleting wifi profile.
requestIdstringYesA unique identifier for the request, allowing tracking and debugging of the operation.
wifiProfileInfoobjectNo
wifiProfileInfo.essidstringNoSSID of the wifi profile to be deleted

Request Examples

Delete wifi profile

{
  "command": "delete_wifi_profile",
  "requestId": "abc123",
  "wifiProfileInfo": {
    "essid": "TestAP2"
  }
}

Response

Response Fields

FieldTypeRequiredDescription
commandstringNo
requestIdstringNo
responseobjectNoStatus of the command execution — check code to confirm success (0) or identify the error.
response.codeintegerYesResponse code indicating success or failure.
  • 0 — Success
  • 2 — Invalid payload
  • 15 — WIFI Error - SSID not found
  • 16 — WIFI Error - Cannot delete active SSID
  • 17 — WIFI Error - SSID missed
response.descriptionstringYesHuman-readable description of the response code (e.g. "Success", "Invalid payload").

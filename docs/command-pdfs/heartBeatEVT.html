ZEBRA TECHNOLOGIES heartBeatEVT — RFD40 / RFD90 IoT Connector API Reference

heartBeatEVT

Command: heartBeatEVT

Description

This event represents a heartbeat signal sent periodically from a device or system to indicate its active status and provide essential metadata like uptime and event details.

Request

Request Fields

No fields available.

Request Examples

No examples available.

Response

Response Fields

FieldTypeRequiredDescription
eventNamestringNoName of the event, indicating it is a heartbeat signal.
timestampstringNoThe timestamp when the event was generated, in ISO 8601 format.
eventNumberintegerNoA unique identifier or sequence number for the event.
upTimestringNoThe uptime duration of the system when the event was generated.
dataobjectNoAdditional data associated with the heartbeat event, referencing different event types.

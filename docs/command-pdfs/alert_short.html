ZEBRA TECHNOLOGIES alert_short — RFD40 / RFD90 IoT Connector API Reference

alert_short

Command: alert_short

Description

Short form of alerts, does not contain details about the alert. This is mainly meant for SOTI.

Request

Request Fields

No fields available.

Request Examples

No examples available.

Response

Response Fields

FieldTypeRequiredDescriptionExample
idstring (enum)YesThe unique identifier for the alert, representing the specific type of event or notification.FIRMWARE_DOWNLOAD_SUCCESS
timestampstringYesThe time when the alert was generated, formatted as a string.
typestring (enum)YesThe category of the message, indicating whether it is a notification or an alert.
prioritystring (enum)YesThe urgency level of the alert, specifying its priority as HIGH, LOW, or CRITICAL.
messageIDstring (enum)YesThe identifier for the message type, used to categorize the source or purpose of the alert.ZEBRA_RFID_HH_ALERTS
descriptionstringYesA brief human-readable text describing the alert or notification.firmware download started

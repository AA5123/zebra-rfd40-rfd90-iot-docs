get_eth

Command: get_eth

Description

This API used to retrieve ethernet configuration details. Security features, static IP configuration, and IPv6 support are not available in the current API version.

Download pdf: 📄 [Download get_eth as PDF](https://aa5123.github.io/zebra-rfd40-rfd90-iot-docs/command-pdfs/get_eth.pdf)

set_operating_mode

Command: set_operating_mode

Description

This operation is used to configure the operating mode of RFID devices. Note: When RFD40/90 sled is configured to FAST_READ profile, data events are not currently supported.

Download pdf: 📄 [Download set_operating_mode as PDF](https://aa5123.github.io/zebra-rfd40-rfd90-iot-docs/command-pdfs/set_operating_mode.pdf)

operatingMode.operatingModes.tagMetaDataToEnable.XPCbooleanNoIndicates whether to report the Extended Protocol Control (XPC) word. XPC provides additional protocol-related information for advanced tags, such as custom memory configurations.
operatingMode.operatingModes.tagMetaDataToEnable.CRCbooleanNoIndicates whether to report the CRC value. CRC is used to verify the integrity of the tag's data and ensure error-free communication.
operatingMode.operatingModes.tagMetaDataToEnable.EPCbooleanNoIndicates whether to report the EPC. EPC is the unique identifier stored in the tag's memory, typically used for item identification.

get_endpoint_config

Command: get_endpoint_config

Description

Command to get configured endpoints.If API does not included endpointName argument, then response includes all active endpoints detailed configuration & list of all saved endpoint names otherwise response includes interested endpoints configuration only

Download pdf: 📄 [Download get_endpoint_config as PDF](https://aa5123.github.io/zebra-rfd40-rfd90-iot-docs/command-pdfs/get_endpoint_config.pdf)

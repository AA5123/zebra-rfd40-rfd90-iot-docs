ZEBRA TECHNOLOGIES mqttConnEVT — RFD40 / RFD90 IoT Connector API Reference

mqttConnEVT

Command: mqttConnEVT

Description

This event provides details about the connection state of a device, including its metadata and protocol versions.

Request

Request Fields

No fields available.

Request Examples

No examples available.

Response

Response Fields

FieldTypeRequiredDescription
connectionStatestring (enum)NoIndicates the connection status of the device. Possible values are 'CONNECTED' or 'DISCONNECTED'.
timestampstringNoThe time at which the MQTT connection event occurred, formatted as HH:MM:SS.
deviceModelstring (enum)NoSpecifies the model of the device involved in the MQTT connection event. Supported models are 'RFD40' and 'RFD90'.
deviceSerialNostringNoThe unique serial number of the device involved in the MQTT connection event.
apiVersionnumberNoThe version of the API in use during the MQTT connection event.
mqttVersionnumberNoThe version of the MQTT protocol being used for the connection.

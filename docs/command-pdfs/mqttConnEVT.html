mqttConnEVT

Command: mqttConnEVT

Description

This event provides details about the connection state of a device, including its metadata and protocol versions.

Request

Request Fields

No fields available.

Request Examples

No examples available.

Response

Response Fields

FieldTypeRequiredDescriptionExample
connectionStatestring (enum)NoIndicates the connection status of the device. Possible values are 'CONNECTED' or 'DISCONNECTED'.CONNECTED
timestampstringNoThe time at which the MQTT connection event occurred, formatted as HH:MM:SS.12:17:56
deviceModelstring (enum)NoSpecifies the model of the device involved in the MQTT connection event. Supported models are 'RFD40' and 'RFD90'.RFD40
deviceSerialNostringNoThe unique serial number of the device involved in the MQTT connection event.RFD40-24190525100354
apiVersionnumberNoThe version of the API in use during the MQTT connection event.1.0
mqttVersionnumberNoThe version of the MQTT protocol being used for the connection.3.1.1

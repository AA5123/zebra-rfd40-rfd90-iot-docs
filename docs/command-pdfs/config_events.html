ZEBRA TECHNOLOGIES config_events — RFD40 / RFD90 IoT Connector API Reference

config_events

Command: config_events

Description

Enables or disables specific device events. To apply the changes, a device reboot is required. For RFD4090 devices, only the following events are supported: terminalConnection, firmwareUpdate, network, exceptions, ntp, heartbeat, power, battery, and fileDownload.
